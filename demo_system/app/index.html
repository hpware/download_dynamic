What is this?

This is a downloading system made like the Massgrave downloading system (which is inspired by the Microsoft Windows download system, but thats beside the point), and this project mimics that, but using Bun, React, Tailwind & Postgres! You can learn more by going to https://yhw.tw/ljads_guide! Bonus, this project is fully open source!
