What is this?

This is a downloading system made like the Massgrave downloading system (which is inspired by the Microsoft Windows download system, but thats beside the point), and this project mimics that, but using Bun, React, Tailwind & Postgres! You can learn more by going to https://yhw.tw/ljads_guide! I also spun up a testing instence for you to test out the app, it is Tailwind & Postgres! You can learn more by going to ljads-demo.yuanhau.com Bonus, this project is fully open source and self hostable to your liking! Psst, if you are too lazy, I've made a Zeabur template for easy deployment!
